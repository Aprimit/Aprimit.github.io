Connected Health Platform

The SAP Connected Health is an extension to the HANA platform that is completely based on XS and therefore without the need for an external application server. The presentation logic runs on the client as a browser-based web application that communicates via HTTP/S. The SAP Connected Health UI is built as a HANA Fiori Launchpad and serves as the central entry point for all the applications that are enabled with the means of SAP Connected Health. SAP Connected Health User Management extends the core HANA user management functionality and provides tools for a User Administrator to onboard users belonging to an organizational unit/clinic.

Connection Server

Connection Server (CS) is the SAP Business Objects data access software that manages the connection between the application and the data source. Connection Server comes with the SAP Business Objects Enterprise (BOE) suite. CS allows SAP Business Objects applications such as the universe design tool (a.k.a. Designer) and SAP Business Objects interactive Analysis (a.k.a. WebI) to connect to and run queries against a data source. Connection Server does not have a user interface. You create and administer connections from the user interface of SAP Business Objects applications, or by editing Connection Server configuration files. You optimize the way data is passed through Connection Server by modifying data access configuration files. These files are in XML format, and are installed with Connection Server. You can set parameter values to apply to a specific data access driver, or to all installed data access drivers.
